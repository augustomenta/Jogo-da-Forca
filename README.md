Jogo da Forca em Java 🕹️
Este é um projeto simples de Jogo da Forca desenvolvido em Java para praticar lógica de programação, manipulação de strings e estruturas de repetição.

🚀 Sobre o Projeto
O objetivo do jogo é adivinhar a palavra secreta escolhida antes que as tentativas se esgotem. O programa solicita uma palavra inicial e, em seguida, permite que o jogador tente adivinhá-la letra por letra.

🧠 O que eu pratiquei neste projeto:
Entrada de Dados: Uso da classe Scanner para capturar as jogadas.

Manipulação de Strings: Uso de métodos como .toUpperCase(), .length() e .charAt() para processar a palavra secreta.

Estruturas de Controle: Implementação de laços while para o fluxo principal e for para percorrer os caracteres da palavra.

Arrays: Criação de um array de caracteres (char[]) para gerenciar o estado oculto da palavra (ex: _ _ _).

Lógica Condicional: Sistema de contagem de tentativas e verificação de vitória ou derrota.

🛠️ Tecnologias Utilizadas
Linguagem: Java 17+

IDE: IntelliJ IDEA 

Paradigma: Programação Estruturada / Orientada a Objetos
